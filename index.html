How EV is calculated
Each raid drop has its own independent roll per run:
EV / run = Σ ( drop_chance × avg_quantity × cheapest_sell_price )
Drop chance = rate_permil ÷ 10000 (e.g. 6000 → 60%). Completion rewards are added on top. Dashed chips = rank reward groups. Hover any chip for breakdown. This is a statistical average across many runs.
